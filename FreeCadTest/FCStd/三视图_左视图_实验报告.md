# 三视图（左视图）实验报告（面向高中生）

## 实验目的

1. 理解左视图（Left view）与三维坐标 (X, Y, Z) 的对应关系。
2. 学会从已打印/实体模型对应的 FreeCAD 文档中读取关键点的三维坐标，并把它们投影到左视图平面上。
3. 训练学生的立体到平面投影能力，提高空间想象力与工程制图能力。

## 实验材料与软件

- 已打印的实体模型（从 `123.FCStd` 导出的或直接打印）。
- 带有模型文件并解压后的目录：`123_extracted`（含 `Document.xml`，另含 Body 的 Shape）。
- FreeCAD（用于打开模型、选择点并运行宏）——本实验使用 FreeCAD GUI 中运行的宏 `extract_left_view_points.FCMacro`。
- Python 3（可选，用于后处理 CSV）。

## 左视图坐标规则（本实验约定）

为避免教材差异，本实验采用两种同时给出的左视图坐标表示：

1. 直接投影（YZ）：把三维点 (X, Y, Z) 投影到 YZ 平面，得到二维坐标 (Y, Z)。
2. 左视图常用记法（镜像处理）：当观察者站在模型的左侧（负 X 方向）朝模型看时，水平轴往右为- Y，因此左视图通常把水平坐标写成 -Y；垂直仍为 Z。即左视图二维坐标为 (-Y, Z)。

实验输出将同时给出两种表示，方便教学比较并讨论正负号的物理含义。

## 实验步骤（教师示范）

1. 在 FreeCAD 中打开 `123.FCStd`（或把 `123_extracted` 中的模型作为项目打开）。
2. 在 FreeCAD GUI 中选择你关心的点：
   - 方法 A（推荐）：在树视图或模型中选择实体（例如 `Body`），宏将提取该实体所有顶点（Vertex）。
   - 方法 B（更精确）：在 Part 工作台中进入顶点选择模式，手动选中若干顶点（Shift + 点击），宏将只读取所选顶点。
3. 在 FreeCAD 的菜单中运行宏 `extract_left_view_points.FCMacro`（将其放入你的 FreeCAD 宏目录或通过 `Tools -> Macros...` 载入并运行）。
4. 宏将在文档目录下生成 `left_view_points.csv`，并在 FreeCAD 控制台输出结果。CSV 包含：序号、X、Y、Z、Left_YZ_X（即 Y）、Left_YZ_Y（即 Z）、LeftView_Horiz（-Y）、LeftView_Vert（Z）。
5. 把 CSV 发给学生或打印，作为练习与答案对照表。

## 课堂练习（建议）

1. 给每位学生一份 CSV（去掉坐标答案列），要求他们根据模型观察手绘左视图，填入每个点的二维坐标。
2. 要求学生对比手绘与 CSV 中的（-Y,Z）结果，讨论差异并解释符号含义。
3. 拓展：让学生选取任意三个点，手动绘制它们的三视图并计算相互距离，验证与三维距离的关系。

## 教学建议与评分

- 正确提取并报告点坐标（含左右视图两种记法）：30%
- 手绘左视图坐标与 CSV 一致（位置与符号）：50%
- 对镜像、观察方向和符号解释清晰：20%

## 备用说明（若模型较复杂）

- 若实体有大量顶点，建议教师先在 FreeCAD 中选取感兴趣的顶点（Sub-element 选择），再运行宏以减少学生题量。
- 宏也会在控制台输出并提示 CSV 路径，便于教师快速检查。
